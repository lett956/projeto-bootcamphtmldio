História da Web

(Voltar)

Nessa aula é citado alguns dos primeiros computadores, alguns dos mais antigos colaboradore também, tanto para computadores quanto para a programação em si.

O que são Clients

(Voltar)

São os usuários dos dispositivos (notebooks, smartphones e computadores desktops), são considerados "clientes" dos servidores. Os dispositivos clientes enviam solicitações de páginas web ou aplicativos aos servidores e os servidores fornecem respostas.

O que são Servers

(Voltar)

Server (no inglês), como o próprio nome indica ao traduzir, significa 'servidor' e um servidor é um computador equipado com um ou mais processadores, bancos de memória, portas de comunicação, softwares e, ocasionalmente, algum sistema para armazenamento de dados, como hard disks internos ou memórias SSD.

Linguagens de Programação

(Voltar)

Há uma(mais) linguagem(ns) computacional(is) que funciona por meio de uma série de instruções, símbolos, palavras-chave, regras semânticas e sintáticas. A linguagem de programação permite que um programador crie programas a partir de um conjunto de ordens, ações consecutivas, dados e algoritmos. E ainda há, inclusive, Linguagens Server-Side e Linguagens Server-Side, como também um top 5 de linguagens mais utilizadas de acordo com RedMonk.
